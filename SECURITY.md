# Security Policy

## Supported Versions

This repository is a WordPress plugin. Only the latest released version is supported with security updates.

## Reporting a Vulnerability

Please **do not** open a public issue for security problems.

Instead, report security issues privately:

- Email: security@apicentraal.nl

Include as much detail as possible:

- Steps to reproduce
- A proof-of-concept (if available)
- Affected WordPress / PHP versions
- Any logs or screenshots that help

We aim to acknowledge reports within **5 business days**.

## Disclosure

We prefer coordinated disclosure. Once a fix is available, we will publish a release note describing the issue at a high level.
